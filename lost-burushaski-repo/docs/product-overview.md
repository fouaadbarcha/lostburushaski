# Product Overview

## Project Name
Lost Burushaski

## Vision
Build a modern digital platform that helps preserve, organize, teach, and expand the Burushaski language through structured dialect separation, strong search, community knowledge building, and a future standard learning layer.

## Strategic Direction
The project should not flatten Hunza, Nagar, and Yasin into one messy pool. Instead, it should:

- preserve the three major dialects separately
- connect them through a shared concept system
- later introduce a curated Standard Burushaski layer for learners

## Why This Matters
Burushaski growth will not happen through raw preservation alone. It needs:

- discoverability
- usability
- standardization for learners
- trust from native speakers
- a system that turns community knowledge into structured learning

## Main User Types

### Learners
- non-native speakers
- heritage learners
- beginners who need a simpler standard learning path

### Native Speakers
- Hunza speakers
- Nagar speakers
- Yasin speakers
- community members who already know the language and want to contribute

### Trusted Contributors and Experts
- strong contributors with high trust scores
- language experts
- native speakers with deep knowledge
- moderators and reviewers

## Core Systems

1. Lexicon system
2. Search system
3. Community question and answer system
4. Rating and trust system
5. Standardization system
6. Learning system
7. Dashboard and moderation system

## Product Positioning
This is not just a dictionary. It should become:

- a living Burushaski lexicon
- a community language question system
- a dialect-aware semantic search engine
- a source for Standard Burushaski
- a learner-focused language course platform
